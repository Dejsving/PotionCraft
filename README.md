# PotionCraft
Проект для работы с правилами Гербализм и Алхимия для DnD 5.0

Правила: https://dnd.su/homebrew/articles/homebrew/109-gerbalizm_i_alkhimiya/?ysclid=lltrfaeclk207962267#spisok_ingredientov

Особенности: 2к6 означает 2 раза бросить кубик к6 (равномерное распределение от 1 до 6) и сложить результаты

Всякий раз, когда ДМ делает бросок на 2-4 или 10-12 на любой территории, ДМ должен бросить к% и если результат выше 75, ингредиент становится Элементальной водой вместо ингредиента который был описан.

Доп. правила таблиц Ингредиенты:
